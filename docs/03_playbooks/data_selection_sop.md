# Data Selection SOP

Select samples that improve coverage without increasing overfitting risk.

Prefer:

- Different plot phases
- Different emotional beats
- Different conflict types
- Clear visible text
- Verified author/series match

Avoid:

- Near duplicates
- Ambiguous authorship
- Samples that rely on comments or hidden context
- Any sample that required risk-control bypass

Do not paste raw body text into committed docs.
