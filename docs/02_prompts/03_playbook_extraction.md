# Prompt: Playbook Extraction

Extract a transferable genre playbook from the provided local sample notes.

Focus on:

- Premise engine
- Reader promise
- Protagonist desire and pressure
- Relationship/conflict structure
- Escalation loop
- Scene rhythm
- Emotional payoff pattern
- Serial cliffhanger pattern
- Common constraints and taboos

Do not:

- Imitate the source author.
- Copy source phrasing.
- Reuse source scenes.
- Analyze title formulas for Experiment 001.

Output a playbook that another writer could use to create original concepts in the same broad category.
