# Prompt: Playbook Extraction

## Purpose

Extract a reusable genre playbook from the samples and audits.

## Prompt

Using the sample dataset, sample coverage audit, and overfitting risk audit, create a reusable genre playbook for original Xiaohongshu-style canary-lit / melodramatic serialized fiction.

Boundary:
- Do not imitate any specific author.
- Do not reproduce source text, specific scenes, names, or distinctive personal expressions.
- Only extract genre-level and platform-level mechanisms.

The sample may contain platform-coded language, euphemisms, abbreviations, or substitute words, such as “米” for money or “灰灰” for gray-market activity.

Please:
1. Preserve the original wording.
2. Infer likely meanings only when context supports it.
3. Create a coded-language glossary with confidence levels.
4. Separate platform-native flavor from risky euphemisms.
5. Do not treat all coded terms as reusable style.
6. For risky terms, analyze their narrative function but do not generate operational or instructional details.

Please output:

1. Genre definition
2. Reader desire map
3. Emotional engine
4. Character relationship model
5. Power imbalance model
6. Canary-lit atmosphere model
7. Opening hook formulas
8. Mid-story escalation formulas
9. Cliffhanger formulas
10. Platform-native expression patterns
11. Trope library
12. Common failure modes
13. Originality boundaries
14. Writing rubric
15. Prompt templates for later generation

Important:
Since the source work does not have independent post titles, mark title formula as out of scope for this playbook version.
