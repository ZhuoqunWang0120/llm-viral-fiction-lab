# Prompt: Playbook Stress Test

Stress-test the playbook for transferability.

Tasks:

1. Identify rules that are too source-specific.
2. Identify rules that are too vague to guide writing.
3. Propose safer, more general versions.
4. Generate three original premise directions that follow the playbook without reusing source text, source scenes, or author-specific expression.

Output:

- Weak rules
- Revised rules
- Original premise directions
- Overfitting warnings
