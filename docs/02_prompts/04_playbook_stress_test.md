# Prompt: Playbook Stress Test

## Purpose

Test whether the playbook can generate original concepts without overfitting to the source work.

## Prompt

Do not write full fiction yet.

Use the playbook to generate 8 original serialized fiction concepts.

Each concept should include:

1. Series name
2. One-sentence hook
3. Core canary relationship
4. Female lead
5. Male lead
6. Core power imbalance
7. First-post opening hook
8. First 5-post conflict trajectory
9. Viral potential score, 1–10
10. Originality score, 1–10
11. Sustainability score, 1–10
12. Main risk

After generating the concepts, evaluate:

1. Does the playbook produce concepts with the right genre flavor?
2. Are the concepts too close to the source work?
3. Which concepts are strongest?
4. Does the playbook need revision?
5. Is more data needed?
