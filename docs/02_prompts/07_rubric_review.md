# Prompt: Rubric Review

Evaluate the draft against the writing rubric.

Score:

- Hook clarity
- Premise freshness
- Genre fit
- Emotional engine
- Character pressure
- Scene momentum
- Serial payoff
- Originality / non-imitation
- Platform readability

Output:

- Scores
- Strengths
- Weaknesses
- Revision priorities
- Overfitting concerns
