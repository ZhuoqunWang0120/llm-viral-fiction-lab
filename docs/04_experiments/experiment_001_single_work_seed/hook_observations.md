# Hook Observations

## Purpose

This file tracks hook-related observations that emerge from playbook and concept-generation experiments.

Experiment 001 marks full title formula analysis as out of scope because the source work does not have independent post titles. However, later generated artifacts revealed another useful hook type: in-series hook lines.

## Observation: In-Series Hook Lines

Some generated character lines create strong continuation desire for readers who are already following the series.

Example:

> 大佬说：这个模型本来就是为你训练的。

These are not necessarily suitable as first-post cold-start titles. They are better understood as:

- episode ending hooks,
- next-post opening lines,
- in-series teasers,
- comment-bait lines,
- emotional cliffhanger lines.

## Working Taxonomy

| Hook Type | Function | Current Status |
|---|---|---|
| Cold-start title hook | Attract new readers who have not seen the series | Out of scope for Experiment 001 |
| In-series continuation hook | Make existing readers click the next episode | Emerging mechanism |
| Ending quote hook | End a post with a memorable character line | Strong candidate |
| Episode teaser line | Used in intro/preview to frame the next conflict | Strong candidate |
