# Single-Work Seed Playbook v0: 金丝雀文学机制假设版

> Experiment 001: Single-Work Seed Dataset  
> Source basis: `chatgpt_input.md` + `experiment_001_sample_coverage_audit.md`  
> Dataset: 13 条同一作者、同一部 RedNote/Xiaohongshu 金丝雀文学连载 seed samples  
> Status: 机制假设版，不是通用品类定论

---

## 1. Playbook Scope and Disclaimer

### 1.1 Scope

本文档用于总结 **Experiment 001** 中 13 条单作品 seed samples 所呈现出的叙事机制，目标是形成一个可用于后续原创开发、样本扩充和跨作品验证的 **机制假设版 playbook**。

本文档关注：

- 金丝雀文学的情绪引擎
- 金丝雀感如何被制造
- 权力差如何进入人物关系
- 物质细节如何服务剧情
- 平台化表达如何增强阅读黏性
- 原创生成时应保留哪些机制、避免哪些高过拟合特征

### 1.2 Hard Boundaries

本 playbook 有以下边界：

- 这是基于 13 条单作品 seed samples 的机制假设版。
- 不代表金丝雀文学或网红小说的全量品类规律。
- 不模仿作者本人。
- 不复刻原文句子、桥段、人名、设定或个人化表达。
- 不复用原作人物关系、核心事件、具体场景组合。
- 标题公式分析 out of scope，因为该作品没有每篇独立标题。
- coded language 需要保留原文、单独注释，不要直接替换。
- 风险/灰色表达只分析叙事功能，不生成操作性细节。
- 原始样本只用于本地实验分析，不用于公开发布。

### 1.3 Recommended Use

适合用于：

- 做原创金丝雀文学/名利场文学/阶层博弈短篇连载的前期设定
- 给 LLM 提供“机制级”创作约束
- 设计原创人物关系、原创情节冲突、原创连载节奏
- 后续和其他作品对照，验证哪些机制可迁移

不适合用于：

- 仿写该作者语言
- 改写原作情节
- 复刻原作人物和具体名利场配置
- 生成现实可执行的灰色产业细节
- 直接下结论说“这就是金丝雀文学通用公式”

## 2. 品类机制假设

### 2.1 Core Hypothesis

金丝雀文学的核心不是单纯“美女被有钱人养”，而是：

> 一个年轻、漂亮、有欲望但资源不足的女性，进入由金钱、身份、年龄、权力、信息差共同构成的高压关系场，并在被供养、被展示、被定价、被误判的过程中，逐渐形成自己的生存策略。

这里的“金丝雀”不是纯被动角色。她可以被供养，但她也在观察、计算、选择、忍耐、升级、撤退或反杀。

### 2.2 样本中反复出现的机制

| 机制 | 叙事功能 | 可迁移性 |
|---|---|---:|
| 财富进入亲密关系 | 把爱情变成交易、庇护、投资和风险管理 | 高 |
| 年龄/资源差 | 制造不平等、保护感、控制感和危险感 | 高 |
| 女性策略性成长 | 让角色不是单纯受害者，而是有判断力的行动者 | 高 |
| 物质细节定价人物 | 用房子、表、车、办公室、学费等呈现关系状态 | 高 |
| 名利场局 | 让人物被公开比较、试探、争夺、羞辱 | 高 |
| 男性资源差异化 | 不同男人提供不同价值，也带来不同风险 | 高 |
| 女性之间的对照 | 通过不同女性策略形成命运参照 | 高 |
| 时代背景托底 | 让狗血关系变成社会变迁切片 | 中高 |
| 八卦式旁白 | 提升平台阅读快感 | 中，但作者化风险较高 |
| 现实影射密度 | 增强真实感，但容易过拟合和提高风险 | 低-中 |

### 2.3 品类张力

金丝雀文学的张力来自四个问题：

1. 她到底是被爱，还是被买下？
2. 她到底在享福，还是在承担高风险？
3. 她到底是猎物，还是逐渐成为猎手？
4. 她从这段关系中得到的是身份、钱、人生通道，还是不可逆的代价？

## 3. 读者欲望地图

### 3.1 读者看的不是单一爽点

这类内容的读者欲望通常是复合型的：

| 欲望 | 读者想看到什么 | 对应机制 |
|---|---|---|
| 窥视欲 | 普通人看不到的富豪、二代、名利场、局内规则 | 高密度圈层细节 |
| 爽感 | 女性通过美貌、聪明、判断力获得资源和身份跃迁 | 金丝雀升级线 |
| 残酷感 | 资源关系背后的冷酷、羞辱、风险和代价 | 权力差与灰色边界 |
| 清醒感 | 角色不恋爱脑，能识别资源和风险 | 女性策略模型 |
| 八卦感 | 像听一个“有真实质感”的故事 | 平台化旁白 |
| 阶层想象 | 豪宅、名表、私人场景、名校、海外生活 | 物质细节系统 |
| 替代性判断 | 读者可以在心里评价“她该不该选这个男人” | 男性资源类型对比 |
| 命运沉浮 | 看人物从上桌、失势、翻身、隐瞒到再定价 | 长线连载结构 |

### 3.2 读者情绪曲线

一个有效片段通常会制造以下情绪组合：

1. **好奇**：这是哪个圈层？这个人为什么有资源？
2. **羡慕**：她得到了普通人无法接触的生活。
3. **不安**：这份生活背后明显有代价。
4. **判断**：读者开始评估她的选择是否聪明。
5. **反转**：原来这个男人/资源/关系并不如表面。
6. **追更**：下一个变量出现，读者想知道她如何处理。

### 3.3 核心阅读承诺

> 每一篇都让读者多懂一点名利场规则，也多看到一点人物命运的价格。

## 4. 金丝雀感机制

### 4.1 金丝雀感不是“漂亮”本身

金丝雀感由以下因素共同构成：

| 组成 | 说明 |
|---|---|
| 被供养 | 有人提供房子、钱、出国机会、礼物、生活安排 |
| 被展示 | 她被带入局，被别人看见、比较、评价 |
| 被定价 | 她的美貌、学历、出身、气质、年龄都变成价值指标 |
| 被保护 | 金主提供安全感、遮风挡雨、进入高阶场景的通行证 |
| 被控制 | 资源来源不稳定，选择权不完全在她手里 |
| 被误判 | 外界以为她只是宠物，但她可能在观察和计算 |
| 被替换风险 | 她知道自己不是唯一，也知道关系有生命周期 |
| 自我升级 | 她通过关系获得新的身份、审美、资源和行动能力 |

### 4.2 金丝雀感的三层

#### 第一层：视觉金丝雀

关键词：年轻、美貌、朴素/精致反差、衣着、身材、气质、被注视。

功能：快速建立吸引力。

#### 第二层：生活金丝雀

关键词：房子、车、旅行、餐食、礼物、学习、家庭安置、海外生活。

功能：让读者感受到“被养”的具体质感。

#### 第三层：命运金丝雀

关键词：没有完全自主权、资源依赖、关系风险、身份重塑、离场成本。

功能：让故事不只是炫富，而是有危险和命运感。

### 4.3 好的金丝雀角色需要有“误差”

不要写成单纯拜金，也不要写成纯洁受害者。更有效的是：

- 她有欲望，但不是没脑子。
- 她会享受物质，但也知道物质的价格。
- 她可能真心动情，但仍然保留风险意识。
- 她看似被安排，其实一直在学习规则。
- 她既可能利用别人，也可能被别人利用。

## 5. 权力差模型

### 5.1 权力差的基本公式

> 权力差 = 金钱差 + 年龄差 + 身份差 + 信息差 + 圈层差 + 法律/制度差 + 情感依赖差

### 5.2 权力差维度

| 维度 | 表现 | 剧情作用 |
|---|---|---|
| 金钱 | 谁有现金、资产、学费、房子、礼物 | 决定依赖关系和谈判能力 |
| 年龄 | 年长者拥有经验、历史、前妻/家庭、旧关系 | 制造保护感和不透明感 |
| 身份 | 富豪、二代、名校、金领、网红、掮客 | 决定谁能带谁进场 |
| 信息 | 谁知道真相、黑历史、财务危机、真实资产 | 制造反转和危险 |
| 圈层 | 谁能进入饭局、办公室、豪宅、海外圈 | 制造阶层门槛 |
| 制度 | 婚姻、学位、签证、公司、合同、调查 | 把私人关系连接到社会结构 |
| 情感 | 谁更爱、谁更需要、谁更怕失去 | 让交易关系产生情绪波动 |

### 5.3 权力差的动态变化

权力差不是固定的。好的故事会让权力位置发生变化：

1. 初始：金主强，金丝雀弱。
2. 中段：金主遇到危机，金丝雀掌握信息或替代选择。
3. 反转：金丝雀获得资产、身份、学历、人脉或独立资源。
4. 新局：她开始被其他人重新定价，甚至反过来影响男性命运。

### 5.4 权力差写作原则

- 不要只写“他有钱，她漂亮”。
- 要写清楚：他的钱是否可流动？是否愿意给？给的是礼物、生活费、资产，还是身份通道？
- 要写清楚：她依赖他什么？她有没有替代方案？她知道多少真相？
- 每一次亲密互动最好都带一点权力位置变化。

## 6. 人物关系模型

### 6.1 核心关系类型

| 关系类型 | 叙事功能 |
|---|---|
| 金主 vs 金丝雀 | 核心供养、控制、保护、交易和情感张力 |
| 富一代 vs 年轻女性 | 资源巨大，但历史包袱和风险也大 |
| 富二代 vs 年轻女性 | 有身份幻觉，但未必有现金和兑现能力 |
| 失势二代 vs 金丝雀 | 有残余名望，但资源下滑，适合制造错位感 |
| 创一代 vs 高学历女性 | 钱需要文化背书，女性提供身份装饰 |
| 前妻/原配/旧家庭 vs 新女性 | 展示男性历史、财产分配和替代性 |
| 女性朋友/对照组 | 互相羡慕、比较、提醒、利用、镜像成长 |
| 中介/掮客/机构 | 把亲密关系接入灰色经济和名利场结构 |
| 家族成员 | 把个人选择变成家庭利益和阶层想象 |

### 6.2 关系不是平面图，而是资源流动图

每段关系都应该问：

- 谁给谁钱？
- 谁给谁身份？
- 谁给谁安全感？
- 谁给谁审美包装？
- 谁给谁进入新圈层的机会？
- 谁承担名誉风险？
- 谁掌握对方的秘密？
- 谁可以随时离场？

### 6.3 女性对照关系很关键

样本中的女性不是单一女主视角，而是多个女性策略互相映照。原创时可以保留这种机制：

| 女性类型 | 功能 |
|---|---|
| 被保护型 | 展示被供养、被安置、被宠爱的表面好处 |
| 进取型 | 展示主动资源配置、身份升级、风险分散 |
| 老江湖型 | 展示规则、诱导、利用、介绍局 |
| 前妻型 | 展示上一代女性如何完成资源结算 |
| 家族女性型 | 展示普通家庭如何理解金钱和婚恋 |

## 7. 女性角色策略模型

### 7.1 女性角色不能只有“爱”或“拜金”

有效的女性角色通常有多重动机：

- 想过好日子
- 想进入更高圈层
- 想被爱、被保护、被看重
- 想摆脱原生环境或旧身份
- 想证明自己值得更高价格
- 想规避风险、保留退路
- 想完成身份重塑

### 7.2 策略维度

| 策略 | 说明 | 剧情作用 |
|---|---|---|
| 资源识别 | 判断一个人真正能给什么 | 避免角色恋爱脑 |
| 风险分散 | 不把未来押在单一男性身上 | 制造复杂关系网 |
| 身份包装 | 用学历、审美、气质、故事提高身价 | 服务阶层跃迁 |
| 情绪劳动 | 在男性低谷、焦虑、失意时提供安慰 | 增加亲密感和交换价值 |
| 场合管理 | 知道什么时候说话、沉默、离开、装不懂 | 展示局内生存能力 |
| 退路设计 | 房产、现金、学历、海外身份、人脉 | 支撑后期反转 |
| 黑历史管理 | 处理过去、隐瞒、切割、重塑 | 制造中段危机 |

### 7.3 女性成长弧

一个可迁移的成长弧可以是：

1. 初入局：靠美貌/年轻/偶然机会被带入。
2. 学规则：开始理解钱、身份、关系、局、话术。
3. 被定价：意识到自己在别人眼里是什么价格。
4. 主动选择：不再只被安排，开始挑选资源和风险。
5. 付代价：旧选择留下债务、秘密、关系隐患。
6. 重塑身份：通过学位、资产、事业、婚姻、社交标签变成新的人。
7. 再上桌：不只是被挑选的人，而是能影响局面的人。

### 7.4 避免写成“女性开挂”

策略型女主需要有局限：

- 她可能判断错人。
- 她可能缺现金。
- 她可能被旧合同/旧关系/旧照片困住。
- 她可能被家庭期待裹挟。
- 她可能对某个人真的动情。
- 她可能在某个阶段没有足够谈判权。

## 8. 男性资源类型模型

### 8.1 男性不应只有“有钱/没钱”

更好的模型是把男性拆成不同资源包：

| 类型 | 资源 | 风险 | 剧情用途 |
|---|---|---|---|
| 富一代 | 现金、资产、企业、人脉、海外布局 | 年龄、前史、财务危机、家庭结构复杂 | 提供大资源和大风险 |
| 富二代 | 家族名号、生活方式、圈层入口 | 现金不足、兑现能力弱、情绪脆弱 | 制造幻觉和落差 |
| 失势二代 | 旧权力光环、残余人脉、身份故事 | 资源下滑、报复风险、自尊敏感 | 制造错位和危险感 |
| 创一代 | 新钱、暴富、兑现能力 | 文化焦虑、身份焦虑、审美粗糙 | 需要女性提供文化背书 |
| 书香富豪 | 社会地位、文化资本、稳定形象 | 抠门、规训、要求高、不愿给现金 | 制造“有价值但薅不动”的冲突 |
| 掮客型 | 信息、人脉、局、渠道 | 不稳定、风险外溢、站队成本 | 推动情节进入更大局 |
| 老江湖型 | 规则、判断、介绍、局中局 | 利用他人、免责声明式诱导 | 制造陷阱和转折 |
| 情绪依赖型 | 需要被安慰、崇拜、理解 | 低谷时脆弱，翻身后未必兑现 | 制造情感真伪问题 |

### 8.2 男性资源的关键判断问题

写作时不要只问“他有多有钱”，而要问：

- 他的钱是资产还是现金？
- 他愿不愿意给女主可支配资源？
- 他给的是消费品、身份标签，还是真正安全垫？
- 他现在是在上升期、稳定期、危机期，还是失势期？
- 他需要女主什么：美貌、陪伴、文化背书、情绪安慰、门面、秘密协助？
- 他有没有旧家庭、债务、黑历史、圈层敌人？
- 他对女主的误判是什么？

### 8.3 男性角色最好有“错位”

样本中有效的男性通常不是简单霸总，而是有错位：

- 看似富，其实没有现金。
- 看似落魄，其实还有名望残值。
- 看似爱才女，其实想低成本占有。
- 看似大方，其实只愿意买展示性消费。
- 看似能保护人，其实自己也在被时代吞没。

## 9. 物质细节如何服务剧情

### 9.1 物质细节不是装饰

金丝雀文学中的物质细节要承担剧情功能。它们不是单纯炫富，而是用来回答：

- 这段关系到了什么阶段？
- 男性愿意付出到什么程度？
- 女性是否获得了真正可支配资源？
- 角色的阶层位置是否发生变化？
- 表面体面背后有没有现金危机？

### 9.2 物质细节类型

| 类型 | 示例方向 | 叙事功能 |
|---|---|---|
| 交通 | 私人交通、商务舱、司机、跨城/跨国移动 | 展示圈层入口和行动半径 |
| 住宅 | 公寓、豪宅、海外房子、临时住处 | 展示安置、控制、安全感 |
| 礼物 | 表、包、珠宝、艺术品、衣服 | 展示被宠、被展示、被定价 |
| 现金 | 月薪、学费、违约金、生活费、周转资金 | 决定真实谈判能力 |
| 教育 | 名校、私教、海外项目、学历标签 | 服务身份重塑 |
| 办公空间 | 独立办公室、职位、公司楼层 | 从金丝雀到事业女性的转化 |
| 餐饮 | 高端饭局、私人场所、普通食物反差 | 制造亲密感或讽刺感 |
| 家庭物件 | 书房、钢琴、家族摆设、出版物 | 展示文化资本或身份焦虑 |

### 9.3 物质细节的判断原则

同样是“给钱”，剧情含义不同：

| 给法 | 含义 |
|---|---|
| 给礼物 | 让她被展示、被装饰，不一定给自由 |
| 给房子住 | 给安全感，也可能制造依赖 |
| 给现金 | 提高她的行动自由 |
| 给学费 | 投资她的新身份 |
| 给职位 | 让她获得社会身份，但也可能引发质疑 |
| 给资源入口 | 让她进入新局，也让她承担新风险 |

### 9.4 推荐写法

每个物质细节最好同时服务两件事：

1. 满足读者的阶层窥视欲。
2. 推动人物关系或权力位置变化。

## 10. 开头钩子机制

### 10.1 常见开头类型

| 开头类型 | 功能 | 可迁移做法 |
|---|---|---|
| 年份 + 时代背景 | 立刻给故事真实感和社会坐标 | 用原创时代/行业背景，不复用原作事件 |
| 判断句 | 直接抛出一个尖锐观点 | 用机制性判断，不照搬原句语气 |
| 财富场景 | 用高阶生活细节吸引读者 | 选择新的物质场景 |
| 人物危机 | 角色一出场就在失控边缘 | 让财富和危险同时出现 |
| 对照开场 | 两类人/两种女人/两个男人并置 | 建立读者判断欲 |
| 秘密开场 | 提示某段不能公开的过去 | 只做叙事悬念，不展开操作细节 |

### 10.2 开头的核心任务

一个有效开头需要在 3–5 句话内完成：

- 给出一个圈层入口。
- 暗示钱或身份的差距。
- 让读者知道这里有不为人知的规则。
- 抛出一个人物即将面对的风险。

### 10.3 开头模板，机制级，不可直接当句子使用

```text
[时代/行业/圈层背景] + [一个看似体面的资源场景] + [主角进入该场景] + [她被看见/被比较/被误判] + [隐藏风险出现]
```

或：

```text
[尖锐判断] + [具体人物案例] + [物质细节] + [关系权力差] + [下一步危机]
```

## 11. 中段冲突升级机制

### 11.1 中段不应只是“更多有钱人出现”

中段需要不断改变资源格局。有效升级方式包括：

| 升级方式 | 说明 |
|---|---|
| 金主财务危机 | 原本稳定的供养关系开始不稳定 |
| 旧家庭出现 | 前妻、孩子、原配、父母让关系复杂化 |
| 新男性出现 | 提供替代资源，也制造新风险 |
| 女性黑历史回流 | 过去的选择变成当前危机 |
| 机构/合同/公司介入 | 私人关系被制度化、债务化、公开化 |
| 海外/跨城移动 | 新场景带来身份重塑，也带来失控 |
| 资源从礼物变现金 | 角色开始追求真正可支配资源 |
| 女性之间互相比较 | 让不同策略的结果可见 |

### 11.2 中段冲突的三个层次

#### Layer 1: 情感冲突

谁爱谁？谁骗谁？谁被替代？谁不甘心？

#### Layer 2: 资源冲突

谁给钱？谁不给钱？谁有现金？谁只有名号？谁能带人进场？

#### Layer 3: 身份冲突

她到底是女朋友、金丝雀、太太候选人、事业女性、名校女神、还是局中的商品？

### 11.3 推荐升级节奏

```text
稳定关系 → 新信息 → 资源不稳定 → 替代选项出现 → 旧风险回流 → 主角做策略选择 → 代价显现 → 身份变化
```

### 11.4 中段冲突写作原则

- 每次升级都应该改变人物的可选项。
- 不要只增加狗血，要增加风险和利益结构。
- 新人物最好带来一种新资源，而不是重复已有男性功能。
- 女性角色的选择要有当时语境下的合理性。

## 12. 结尾悬念机制

### 12.1 有效结尾类型

| 结尾类型 | 功能 |
|---|---|
| 新变量结尾 | 新人物、新资源、新危机出现 |
| 问题结尾 | 直接让读者思考某人还剩什么价值 |
| 反讽结尾 | 表面上获得，实际上埋下风险 |
| 信息差结尾 | 读者知道/主角知道/对方不知道的信息不对称 |
| 地点转移结尾 | 去美国、去新城市、进新局 |
| 关系判断结尾 | 主角意识到某人并非她想象中的资源 |
| 命运感结尾 | 某个决定标志人生进入下一阶段 |

### 12.2 结尾的核心功能

每篇结尾最好做到至少一项：

- 留下一个未解决的资源问题。
- 让人物价值被重新评估。
- 暗示下一段关系或场景。
- 暴露一个新风险。
- 让读者产生“她会怎么选”的判断欲。

### 12.3 结尾不要做的事

- 不要每篇都靠极端反转。
- 不要把悬念写成纯噱头。
- 不要为了追更而引入无法兑现的新坑。
- 不要把灰色内容写成操作指南。

## 13. 平台化表达机制

### 13.1 平台化表达的作用

平台化表达不是简单口语化，而是让读者产生：

- “像在听一个懂行的人讲八卦”
- “像在刷一个有连续剧情的社会观察帖”
- “信息密度很高，但阅读负担不重”
- “每篇都能截出一些观点”

### 13.2 样本中的平台化表达特征

| 特征 | 功能 | 迁移建议 |
|---|---|---|
| 中英混用 | 增强都市、金融、海外、平台语感 | 可适度使用，但不要照搬密度 |
| emoji / 符号 | 降低敏感度，增加平台感 | 可少量使用 |
| 缩写/替代词 | 规避敏感，制造圈内感 | 保留原文并注释，不强行翻译 |
| 八卦旁白 | 让故事像局内人口述 | 可迁移机制，不迁移作者句式 |
| 现实感时间线 | 增强可信度 | 换成原创行业/时代背景 |
| 搜索联想感 | 贴近平台推荐机制 | 标题公式本轮 out of scope |
| 犀利判断 | 提供可传播观点 | 谨慎使用，避免复刻作者口吻 |

### 13.3 平台表达的平衡

平台化表达要做到：

- 信息密度高，但段落不宜过重。
- 口吻可以锋利，但不能只剩吐槽。
- 可以有黑话，但要让读者通过语境理解。
- 可以像八卦，但剧情逻辑必须成立。
- 可以有现实感，但原创时不要映射具体真人。

## 14. Coded Language Handling Rules

### 14.1 总原则

coded language 的处理原则：

> 保留原文，单独注释，不直接替换，不展开操作性细节。

### 14.2 为什么不能直接替换

直接替换会带来三个问题：

1. 丢失平台语感。
2. 误解原文规避策略。
3. 把原本含混的灰色表达过度具体化。

### 14.3 Glossary 初版

| 原文形式 | 注释 | 置信度 |
|---|---|---:|
| 米 | 钱、现金、资金 | 高 |
| cash | 现金/可调动的钱 | 高 |
| money | 钱，常与中文混用 | 高 |
| rich | 有钱/富有，带平台化语感 | 高 |
| A11 / A12 | 可能指财富/资产量级，保留不展开 | 中 |
| GLD | 样本语境中与流动资金/灰色金融相关，保留不展开 | 中 |
| HH | 疑似敏感事件或现实事件缩写，保留不展开 | 低-中 |
| red二代 | 有特定权力/政治家庭背景的二代，原文规避 | 高 |
| 失quan | 失去权力/权势的规避写法 | 高 |
| yan | 烟 | 高 |
| 2M | 200 万或 two million，结合语境判断 | 高 |
| 8w / 8个 | 8 万 | 高 |
| million | 百万级金额 | 高 |
| 白P | 疑似“白嫖”的规避表达 | 高 |
| FAKE | 假货 | 高 |
| xx | 被遮挡/规避的敏感词或固定表达 | 高 |
| 不能播 | 暗示敏感、不便公开细说的经历 | 高 |
| 局 | 饭局/资源局/名利场场合 | 高 |
| 大哥 | 有钱、有资源、有权势的男性 | 高 |
| 金主爸爸 | 提供资源或金钱的男性/资方，带调侃语气 | 高 |
| 下海 | 语境中指进入边缘化/灰色变现赛道 | 中高 |
| 灰灰 | 用户提示的潜在词，当前样本中未稳定出现 | 待验证 |

### 14.4 原创生成时的处理规则

- 可以设计“coded language layer”，但不要复用原文特定组合。
- 对高风险词只保留叙事含义，不展开具体做法。
- 不生成规避审核教程。
- 不生成灰色产业操作流程。
- 不把含混缩写强行解释成确定事实。
- 在分析文档里标注“置信度”。

## 15. 可迁移机制

以下机制可以迁移到原创作品中，但必须换人物、换场景、换事件、换表达：

| 可迁移机制 | 原创化方式 |
|---|---|
| 复合权力差 | 换成新行业、新城市、新制度背景 |
| 金丝雀从被动到策略化 | 创造新的成长路径，不复用原作人物弧 |
| 物质细节定价关系 | 换成原创物件和场景 |
| 男性资源类型差异 | 设计不同资源包的原创男性角色 |
| 女性对照组 | 设计不同女性策略，不复刻烧烧/北大姐结构 |
| 新变量结尾 | 每篇结尾引入新资源、新风险、新误判 |
| 时代背景托底 | 用原创行业周期、平台变化、城市变迁 |
| 情绪混合 | 保留“爽感 + 残酷 + 清醒 + 八卦”的组合 |
| 身份重塑 | 从被展示对象到可独立定价的人 |
| 信息差反转 | 让读者、女主、男性掌握不同信息 |

## 16. 不应迁移 / 高过拟合风险特征

### 16.1 不应迁移的内容

| 高风险项 | 原因 |
|---|---|
| 原作人物名 | 直接复刻原作 |
| 原作具体关系链 | 会变成改写而非原创 |
| 原作具体桥段组合 | 如私人飞机 + 某食物 + 某情绪转折这类组合不应复用 |
| 原作标志性判断句 | 容易模仿作者个人化表达 |
| 原作现实影射密度 | 可能带来事实、隐私和安全风险 |
| 原作具体 coded language 组合 | 可分析，不应照搬成风格签名 |
| 固定标题格式 | 本轮 out of scope |

### 16.2 作者化风险较高的特征

| 特征 | 风险 |
|---|---|
| 犀利社会讽刺旁白 | 容易变成仿作者声音 |
| 现实财经/名利场知识密集铺陈 | 可能是该作品独特壁垒 |
| 中英混用的具体节奏 | 过度复用会像原作语气 |
| 对人物的尖锐定性 | 容易复刻原作价值判断方式 |
| 特定阶层黑话密度 | 可能让文本过度像样本 |

### 16.3 安全边界

- 可以写“风险存在”，不要写“如何规避风险”。
- 可以写“机构/合同/黑历史形成压力”，不要写具体勒索或灰产操作。
- 可以写“人物通过局进入名利场”，不要写现实组织方式和操作步骤。
- 可以写“金钱和身份交换”，不要写具体违法交易设计。

## 17. 原创生成约束

### 17.1 Must Do

原创生成时必须做到：

- 使用全新人物名。
- 使用全新城市、行业、机构、家庭背景组合。
- 使用原创关系链。
- 使用原创物质细节。
- 使用原创冲突。
- 保留机制，不保留桥段。
- 保留“权力差 + 女性策略 + 物质定价 + 情绪推进”的结构。
- 明确每一集的剧情功能。

### 17.2 Must Not Do

原创生成时不得：

- 复刻原作人物名、人设和关系。
- 复刻原作具体桥段。
- 复刻原作标志性句子或判断。
- 使用现实可识别人物作为影射对象。
- 生成灰色产业操作细节。
- 把 coded language 变成规避审核教程。
- 让文本只是“有钱人八卦”，缺少人物选择和代价。

### 17.3 原创故事设计检查表

在生成前，先回答：

| 问题 | 目的 |
|---|---|
| 女主缺什么资源？ | 确定进入关系的动机 |
| 男主/资源方能给什么？ | 确定权力差 |
| 他给的是消费品、现金、身份，还是通道？ | 确定资源质量 |
| 女主误判了什么？ | 制造成长空间 |
| 男方误判了什么？ | 制造反转空间 |
| 第三方女性如何映照女主？ | 增加复杂度 |
| 中段危机来自哪里？ | 避免只靠撒狗血 |
| 结尾的新变量是什么？ | 保证追更 |

### 17.4 Prompt Constraint Block

可复用给 LLM 的约束块：

```markdown
请基于“金丝雀文学机制假设”生成原创内容。只迁移机制，不模仿任何特定作者。不得复刻样本中的原文句子、桥段、人名、设定、人物关系或个人化表达。标题公式 out of scope。coded language 如出现，应保留原文并单独注释，不要直接替换。灰色/风险表达只用于叙事功能，不生成操作性细节。重点写清：权力差、资源类型、女性策略、物质细节的剧情功能、每篇结尾的新变量。
```

## 18. 写作 Rubric v0

### 18.1 单篇评分表

| 维度 | 1 分 | 3 分 | 5 分 |
|---|---|---|---|
| 开头钩子 | 只是背景说明 | 有人物/财富/危机之一 | 3 句话内建立圈层、权力差和风险 |
| 金丝雀感 | 只有漂亮/被宠 | 有物质供养和被展示 | 同时有供养、定价、依赖、风险和自我意识 |
| 权力差 | 只是有钱 vs 没钱 | 有钱和身份差 | 金钱、身份、信息、圈层、情感多维交织 |
| 女性策略 | 女主被动接受 | 偶尔判断 | 持续识别资源、风险和退路 |
| 男性资源类型 | 单一霸总 | 有优缺点 | 资源包清晰且有错位 |
| 物质细节 | 堆奢侈品 | 物质体现阶层 | 物质推动剧情和权力变化 |
| 中段冲突 | 纯狗血 | 有关系变化 | 资源格局和身份位置发生变化 |
| 结尾悬念 | 只停在情绪 | 有未解问题 | 新变量改变下一篇局面 |
| 平台表达 | 平淡叙述 | 有口语和热词 | 有信息密度、八卦感、判断欲，但不仿作者 |
| 原创性 | 明显像样本 | 有部分新设定 | 人物、场景、桥段全新，只迁移机制 |
| 安全边界 | 有操作性灰色细节 | 有风险但略具体 | 只分析叙事功能，不提供操作路径 |

### 18.2 单篇最低通过标准

一篇原创样稿至少应满足：

- 有明确权力差。
- 有至少一个物质细节推动剧情。
- 女主有一次策略性判断。
- 男性资源不是单纯“有钱”。
- 结尾出现新变量。
- 没有复刻原作句子、人物、桥段。

### 18.3 连载前 5 篇检查标准

| 篇数 | 功能 |
|---|---|
| 第 1 篇 | 建立女主欲望、缺口、初始场景、圈层入口 |
| 第 2 篇 | 建立资源方的吸引力和危险性 |
| 第 3 篇 | 展示物质供养，但埋下不稳定因素 |
| 第 4 篇 | 引入女性对照或男性替代资源 |
| 第 5 篇 | 第一次让女主意识到自己被定价/误判 |
