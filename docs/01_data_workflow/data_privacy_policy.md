# Data Privacy Policy

Raw collected text is local only. Do not commit it.

Never commit:

- `storage_state.json`
- cookies, tokens, or browser profiles
- `xhs_posts.jsonl`
- `sample_posts.jsonl`
- `dry_run_candidates.jsonl`
- `chatgpt_input.md`
- `chatgpt_sample_input.md`
- `run_log.md`
- `failed.json`
- debug screenshots or page HTML
- files under `data/raw/` or `data/processed/`

Public examples must be synthetic mock data.
